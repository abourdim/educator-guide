# 📖 تربية الأولاد في الإسلام — Tarbiyat al-Awlad

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

> ﴿ وَالَّذِينَ يَقُولُونَ رَبَّنَا هَبْ لَنَا مِنْ أَزْوَاجِنَا وَذُرِّيَّاتِنَا قُرَّةَ أَعْيُنٍ وَاجْعَلْنَا لِلْمُتَّقِينَ إِمَامًا ﴾

منصة تعليمية تفاعلية ثلاثية اللغات (عربي / English / Français) مستلهمة من كتاب **«تربية الأولاد في الإسلام»** للشيخ **عبد الله ناصح علوان** رحمه الله.

An interactive trilingual educational platform inspired by **"Raising Children in Islam"** by Sheikh **Abdullah Nasih Ulwan** (1928–1987).

## 🤖 لمن يريد إعادة بناء التطبيق بالذكاء الاصطناعي — Rebuild with AI

هذا التطبيق تم بناؤه بالكامل باستخدام الذكاء الاصطناعي من خلال المحادثة التالية. إذا أردت إعادة بنائه أو تخصيصه، إليك الخطة والتعليمات:

This app was built entirely using AI through conversation. If you want to rebuild or customize it, here's the prompt plan:

### The Prompt Plan

```
1. RESEARCH PHASE
   - Search for detailed summaries of "تربية الأولاد في الإسلام" by Abdullah Nasih Ulwan
   - Extract: 3 sections structure, 7 responsibilities, 5 methods, age stages
   - Gather Quranic verses and hadith references

2. CONTENT STRUCTURE
   - 8 sections: Home, Lessons (7), Methods (5), Scenarios (5), 
     Assessment (10), Age Roadmap (4), Problem Solver (6), About
   - Full trilingual: Arabic, English, French
   - All content in a single JS translations object

3. DESIGN DIRECTION
   - Islamic aesthetic: gold/teal palette, geometric patterns, Amiri calligraphy font
   - 3 themes: Mosque Gold (dark), Light, Zellige
   - Inspired by github.com/abourdim/hajj-guide for:
     * Sacred bar (bismillah + ayah) at top
     * SVG flag language switches
     * Workshop-diy.org golden "W" logo badge
     * Minimal footer (just "Powered by")
     * Splash screen with countdown

4. TECH CONSTRAINTS
   - Pure HTML + CSS + JS (no framework, no build)
   - Single-file per type (index.html, styles.css, app.js)
   - Offline-capable (only Google Fonts external)
   - localStorage for assessment persistence
   - Web Audio API for sound effects
   - RTL/LTR switching
   - Mobile-first responsive
   - Print styles

5. ITERATE
   - Test every button, section, language switch, theme
   - Simplify what's redundant (footer, splash)
   - Keep sacred text always in Arabic
```

### Example Starting Prompt

```
Build an interactive trilingual (AR/EN/FR) educational web app 
based on the book "تربية الأولاد في الإسلام" by Abdullah Nasih Ulwan.

Include:
- 7 lessons (the 7 responsibilities from the book)
- 5 educational methods (timeline)
- 5 moral scenario quizzes with feedback
- 10-item daily self-assessment with localStorage
- 4-stage age roadmap (0-3, 3-7, 7-10, 10-14)
- 6 problem solver cards
- 3 themes, sound effects, RTL support
- Splash screen with bismillah
- Sacred bar with ayah always in Arabic

Use pure HTML/CSS/JS, no frameworks. 
Islamic aesthetic with gold/teal palette.
Separate files: index.html, styles.css, app.js.
```
